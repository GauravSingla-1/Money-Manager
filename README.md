# Money-Manager
This is a Money Manager project made with the help of Python's GUI (Tkinter) and MySQL as a database. In this user has options to create a new account if doesn't exist or the user can log inÂ with their credentials and can perform three tasks: 1) withdraw money 2) deposit Money, 3) view personal details. All the data is fetched from MySQL Database and the database is updated in real-time.
